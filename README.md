# Galib's Fruit Cutter 🍉🍎🥝

Welcome to **Galib's Fruit Cutter**, an exciting and fast-paced web-based fruit-slicing game built with React, Vite, and Tailwind CSS! 

## 🎮 Game Details & Usage

In this game, your objective is to slice as many fruits as possible while avoiding the obstacles/bombs. 

### How to Play:
- **Swipe or Click & Drag** your mouse/finger across the screen to slice the flying fruits.
- **Earn points** for every successful slice.
- Try to beat your highest score!

### Running Locally:
To run the game on your local machine:
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
   *(Note: The project uses `vite-plugin-singlefile`, so the build will output a single inline `index.html` file in the `dist` folder, which is perfect for platforms like Facebook Instant Games and GitHub Pages.)*
